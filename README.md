Proyecto: Pruebas de API con Postman
En este repositorio documento mi aprendizaje de consumo de APIs desde cero.

Lo que aprendí:
GET: Consultas filtradas por userId.

POST: Creación de nuevos registros con objetos anidados.

PATCH: Actualización parcial de recursos existentes.

Status Codes: Identificación de respuestas 200, 201 y 404.

Herramientas usadas: Postman, JSONPlaceholder API.
